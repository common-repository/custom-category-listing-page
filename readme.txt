Custom Category Listing Page Allow to List Category Posts for each Category by Order ex: `[post_listing]`

== Description ==

Custom Category Listing Page is Use for the Listing Post And Listing Woocommerce Product By Category wise. You can also Shorting Listing Post Or Product By Ordring Date, Title etc. By Using this Shortcode :

`[post_listing]`

**New Features of v 2.0.5**

* You can set Number of post which you can display on listing page.
* Add Pagination Feature in Post Listing Page.
* New Template Design Added

**Customization**

For Listing Post Page Customization you Can Change File custom-category-listing-page/template/default.php

== How does it work? ==

1. Install the plugin
2. Create Page And Add this Shortcode [post_listing].
3. Select Post And Product Categories From Right side of Edit Page.
4. Also Select Ordering Of Categories.
3. And done. You will be able to see List Of Category Which You have been Select. Please refer Screenshot.

If you're not using any of the supported plugin and theme, you can write the custom CSS to apply the fonts.

== Installation ==

The simplest way to install is to click on 'Plugins' then 'Add' and type 'Custom Category Listing Page' in the search field.

= For installation 1: =

1. Login to your website and go to the Plugins section of your admin panel.
2. Click the Add New button.
3. Under Install Plugins, click the Upload link.
4. Select the plugin zip file (custom-category-listing-page.zip) from your computer then click the Install Now button.
5. You should see a message stating that the plugin was installed successfully.
6. Click the Activate Plugin link.

= For installation 2: =

1. You should have access to the server where WordPress is installed. If you don't, see your system administrator.
2. Copy the plugin zip file (custom-category-listing-page.zip) up to your server and unzip it somewhere on the file system.
3. Copy the "custom-category-listing-page" folder into the /wp-content/plugins directory of your WordPress installation.
4. Login to your website and go to the Plugins section of your admin panel.
5. Look for "custom-category-listing-page" and click Activate.
